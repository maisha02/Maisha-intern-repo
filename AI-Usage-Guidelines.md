# AI Usage Guidelines

## AI Tools for My Role
I mainly use tools like ChatGPT and GitHub Copilot to help with coding, debugging, and writing. They are useful when I get stuck or need help understanding something quickly.

## Benefits and Risks
AI can save time and help generate ideas, but it is not always accurate. If used carelessly, it can lead to mistakes or sharing sensitive information.

## Sensitive Information
I should never enter private information like passwords, API keys, or personal data into AI tools. This helps protect both company and user data.

## Validating AI Output
I need to review and test anything AI gives me before using it. I can also check other sources to make sure the information is correct.

---

## Reflection

## When to Use AI
I use AI when I need help understanding concepts or fixing small issues. For final work and decisions, I rely on my own understanding.

## Avoiding Over-Reliance
To avoid relying too much on AI, I try to understand the solution instead of copying it. I make sure I can explain what the code is doing.

## Data Privacy
I will avoid sharing any sensitive or private information with AI tools. I will only use safe and general inputs.

## AI Task

I had to edit the AI responses to match the requirements and fix small mistakes. This showed me that AI is helpful but not perfect.
