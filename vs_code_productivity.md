# VS Code Productivity

# Top 10 Shortcuts

## Navigation
1. Open file – Cmd+P / Ctrl+P  
2. Search in files – Cmd+Shift+F / Ctrl+Shift+F  
3. Go to line – Ctrl+G  
4. Go to definition – F12  

## Editing
5. Add cursor – Option+Click / Alt+Click  
6. Add cursor up/down – Option+Cmd+↑/↓ / Ctrl+Alt+↑/↓  
7. Select next word – Cmd+D / Ctrl+D  
8. Select all same words – Cmd+Shift+L / Ctrl+Shift+L  
9. Move line – Option+↑/↓ / Alt+↑/↓  
10. Comment line – Cmd+/ / Ctrl+/  

## Debugging
- Start debugging – F5  
- Step over – F10  
- Toggle breakpoint – F9  

## Terminal
- Open terminal – Ctrl + `  
- Command palette – Cmd+Shift+P / Ctrl+Shift+P  

## Reflection

The most useful shortcuts for me are search and multiple cursors. They help me find things quickly and edit many lines at once.

Keyboard shortcuts improve productivity because they reduce mouse use and save time. This helps me stay focused and work faster.
