# Kanban Workflow

## How Kanban Works
A Kanban board shows tasks as cards moving through stages. It helps track progress and makes work visible to the whole team.

## Columns in Kanban
Columns represent stages like Backlog, In Progress, Blocked, and Done. Each column shows the current status of tasks.

## Task Movement
Tasks move from one column to another as work progresses. Team members update their own tasks to reflect current status.

## Limiting WIP
Limiting work in progress helps avoid overload and keeps focus on completing tasks. It improves efficiency and reduces unfinished work.
