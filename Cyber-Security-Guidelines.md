# Cyber Security Guidelines

## Common Threats
Common threats in remote work include phishing emails, weak passwords, unsecured Wi-Fi, and malware. These can lead to data leaks or unauthorized access.

## Best Practices
Use strong passwords, keep software updated, avoid public Wi-Fi, and install security updates regularly. Always log out of accounts when not in use.

## Locking Devices
Locking your computer prevents unauthorized access when you step away. It protects sensitive company and user data from being exposed.

## Phishing and Suspicious Links
As Jerm mentioned in the onboarding session, we should not click unknown links or download attachments from untrusted sources. Always verify the sender before taking action.

## Strong Passwords
A strong password is long, unique, and uses a mix of characters. Using a password manager helps store and manage passwords securely.

## Two-Factor Authentication (2FA)
2FA adds an extra layer of security by requiring a second verification step. It should be enabled on all important accounts.
