# AI Tools Experience

## Which AI tools did I try?
I have been using GitHub Copilot for a while.

## What worked well?
It helps me write code faster by suggesting full lines and sometimes full functions.
It really useful for repetitive code or common patterns.
Over the time, I learned how to guide it better by writing clear comments.

## What didn’t work well?
Sometimes the suggestions are not fully correct for the logic i want to implement.
I still need to review and test everything mostly.
It can also suggest code that looks right but doesn't really solve the exact problem.

## When is AI most useful for coding?
AI is most useful for a quick starting point(this could be gathering knowledge about something), writing repetitive code, or learning something new.
It saves time, but we have to make sure to check and understand the code before using it.
