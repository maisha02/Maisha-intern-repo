# React + Tailwind Setup

This project was created using Vite with React.

Steps used to set up the environment:

1. Created a React project using Vite.
2. Installed project dependencies.
3. Installed Tailwind CSS and the Vite Tailwind plugin.
4. Added the Tailwind plugin to vite.config.js.
5. Imported Tailwind in src/index.css.
6. Started the development server and verified Tailwind works.

To run the project:

npm install
npm run dev
